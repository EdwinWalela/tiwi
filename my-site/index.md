
# Tiwi
![tiwi-logo](https://raw.githubusercontent.com/EdwinWalela/tiwi/main/docs/tiwi-mini.png)

---

Generate HTML from markdown

## Getting Started

### Supported markdown syntax

- #- h1
- ##- h2 
- ###- h3
- ![alt](src)- img
- --- - div
- []() - link

---

### Generate HTML

./tiwi build
